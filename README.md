
## Local Development Setup


1. clone the repo

```
git clone https://github.com/Aniket897/drive
```

2. move to working dir

```
cd drive
```

3. client setup
```
cd client
npm install
npm run dev

```

3. server setup
```
cd server
npm install
nodemon
```
